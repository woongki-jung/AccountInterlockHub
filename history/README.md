# history — IA 노드별 이력 인덱스

이 폴더는 완료 작업의 누적 이력을 **IA 노드별 시계열**로 관리한다. 양식·갱신 정책의 단일 출처(정본)는 [`../ai/strategies/ia-history.md`](../ai/strategies/ia-history.md) 이며, 본 폴더는 그 정책에 따른 실제 이력 파일을 담는다. 파일 구조는 노드별 이력 `<ia-code>.md`(1 파일 = 1 IA leaf 노드), 횡단(공통) 사양 이력 `common.md`(영향 IA 열 포함), 연도별 백업 `bak-<YYYY>.md`(현재+직전 연도 밖 entry 회전 보관)로 구성된다. 각 파일 본문은 단일 표(최신순 — 신규 entry 를 맨 위)이며, 산출물 commit 직전 작업 서브에이전트가 상태 🚧(진행)/ℹ️(공통 참조)로 entry 를 추가하고, 평가가 확정되면 평가 주체가 해당 row 의 상태만 ✅(완료)/❌(실패·보완)/⏸️(보류)로 갱신한다(새 row 를 만들지 않는다).
