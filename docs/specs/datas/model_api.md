# API·상태·배치 도메인 데이터 모델 정의

## 개요

- **모델 목적**: 처리 상태와 서비스 대면 조회 API·보관 배치가 다루는 모델을 정의한다. 처리 상태(MDL-301)·처리상태 조회 응답(MDL-302)·배치 실행 결과(MDL-402).
- **관련 서비스**: SVC-005(상태 저장), SVC-006(조회 API), SVC-007(배치).

> 조회 응답(MDL-302)에는 상태 4항목만 담고 회원 키·개인정보·구성 참조를 포함하지 않는다(SEC-005-02).

---

## MDL-301 처리 상태

### 기본 정보

| 항목 | 내용 |
|------|------|
| 모델명 | 처리 상태(ProcessStatus) |
| 분류 | 공통(COM) |
| 사용 서비스 | SVC-005, SVC-006, SVC-007 |
| 매핑 엔터티 | ENT-004 |
| 사용 PROC | PROC-401(저장), PROC-301(조회·갱신), PROC-402(삭제 대상) |
| 용도 | 도메인 모델 |
| 관련 IA 항목 | BAT-01, API-01, BAT-02 |

### 속성 정의

| 속성명 | 데이터 타입 | 필수 | 기본값 | 유효성 규칙 | 마스킹 규칙 | 설명 |
|--------|-----------|------|--------|-------------|-------------|------|
| requestKey | string(UUID v4) | Y | - | UUID v4 형식 | - | 요청 키값(조회 키) |
| configId | string(UUID) | Y | - | 유효 구성 참조 | - | 연동 구성 참조(개인정보 아님) |
| isSuccess | boolean | Y | - | - | - | 처리 성공 여부 |
| isResultConfirmed | boolean | Y | false | - | - | 결과 확인 여부 |
| processedAt | string(ISO8601) | Y | - | - | - | 처리 일시 |
| resultConfirmedAt | string(ISO8601) \| null | N | null | 확인 여부와 정합(CHECK) | - | 결과 확인 일시 |

> 회원 키·개인식별 속성은 두지 않는다(DATA-001-02·DATA-003-01). configId 는 비개인 운영 참조다(§spec-datas.md 담당자 확정 대기 — DATA-003 명시 항목 리비전 연계).

### 엔터티 매핑

| 모델 속성 | 엔터티(ENT) | 엔터티 속성 | 변환 지점 | 변환 규칙 |
|-----------|-------------|-------------|-----------|-----------|
| requestKey | ENT-004 | request_key | 도메인→ENT / ENT→도메인 | 직접 매핑 |
| isSuccess | ENT-004 | is_success | 도메인→ENT | 전달 결과 반영(성공/실패·미전달) |
| isResultConfirmed | ENT-004 | is_result_confirmed | 도메인→ENT | 최초 조회 성공 시 true |
| resultConfirmedAt | ENT-004 | result_confirmed_at | 도메인→ENT | 최초 조회 성공 시 시각 기록 |

### 사용처

| SVC 코드 | 기능 | 용도 | 사용 PROC | 비고 |
|----------|------|------|----------|------|
| SVC-005 | 상태 저장 | 도메인 | PROC-401 | 성공·실패·거부 모두 1건 |
| SVC-006 | 상태 조회·갱신 | 도메인 | PROC-301 | 최초 조회 시 확인 갱신 |
| SVC-007 | 삭제 대상 | 도메인 | PROC-402 | 완료/미완료 90일 경과 |

### 구현 가이드

- 상태 저장 항목은 {요청 키값, 구성 참조, 성공 여부, 결과 확인 여부, 처리 일시, 결과 확인 일시}로 한정하고 그 외 개인정보성 컬럼을 추가하지 않는다. 결과 확인 갱신은 멱등하게 설계한다.

## MDL-302 처리상태 조회 응답

### 기본 정보

| 항목 | 내용 |
|------|------|
| 모델명 | 처리상태 조회 응답(ProcessStatusResponse) |
| 분류 | 서비스(SVC) |
| 사용 서비스 | SVC-006 |
| 매핑 엔터티 | ENT-004(부분 — 4항목) |
| 사용 PROC | PROC-301 |
| 용도 | 응답 모델(서비스 A 대면) |
| 관련 IA 항목 | API-01 |

### 속성 정의

| 속성명 | 데이터 타입 | 필수 | 기본값 | 유효성 규칙 | 마스킹 규칙 | 설명 |
|--------|-----------|------|--------|-------------|-------------|------|
| requestKey | string(UUID v4) | Y | - | - | - | 조회 대상 요청 키값(에코) |
| isSuccess | boolean | Y | - | - | - | 처리 성공 여부 |
| isResultConfirmed | boolean | Y | - | - | - | 결과 확인 여부 |
| processedAt | string(ISO8601) | Y | - | - | - | 처리 일시 |
| resultConfirmedAt | string(ISO8601) \| null | N | null | - | - | 결과 확인 일시 |

> 회원 키·개인정보·configId 를 응답에 포함하지 않는다(SEC-005-02, 상태 4항목만 + 요청 키값 에코).

### 엔터티 매핑

| 모델 속성 | 엔터티(ENT) | 엔터티 속성 | 변환 지점 | 변환 규칙 |
|-----------|-------------|-------------|-----------|-----------|
| isSuccess | ENT-004 | is_success | ENT→도메인 / 도메인→응답 | 직접 매핑 |
| isResultConfirmed | ENT-004 | is_result_confirmed | ENT→도메인 / 도메인→응답 | 갱신 후 반영 |
| processedAt | ENT-004 | processed_at | ENT→응답 | ISO8601 직렬화 |
| resultConfirmedAt | ENT-004 | result_confirmed_at | ENT→응답 | ISO8601 직렬화(null 허용) |

### 사용처

| SVC 코드 | 기능 | 용도 | 사용 PROC | 비고 |
|----------|------|------|----------|------|
| SVC-006 | 응답 변환 | 응답 | PROC-301 | 미존재 시 404, 회원 키 미포함 |

### 구현 가이드

- 응답 DTO 에 회원 키·configId 필드를 두지 않는다. 마스킹은 응답 DTO 변환·로그 포맷터 계층에서 일괄 적용한다.

## MDL-402 배치 실행 결과

### 기본 정보

| 항목 | 내용 |
|------|------|
| 모델명 | 배치 실행 결과(BatchRunResult) |
| 분류 | 서비스(SVC) |
| 사용 서비스 | SVC-007 |
| 매핑 엔터티 | 없음(전이 요약 — ENT-006 감사 로그 detail 로 기록) |
| 사용 PROC | PROC-402 |
| 용도 | 도메인/결과 모델(배치 요약) |
| 관련 IA 항목 | BAT-02 |

### 속성 정의

| 속성명 | 데이터 타입 | 필수 | 기본값 | 유효성 규칙 | 마스킹 규칙 | 설명 |
|--------|-----------|------|--------|-------------|-------------|------|
| targetCount | number | Y | 0 | >= 0 | - | 삭제 대상 건수 |
| deletedCount | number | Y | 0 | >= 0 | - | 실제 삭제 건수 |
| elapsedMs | number | Y | 0 | >= 0 | - | 소요 시간(ms) |
| runAt | string(ISO8601) | Y | - | - | - | 배치 실행 시각 |

### 엔터티 매핑

| 모델 속성 | 엔터티(ENT) | 엔터티 속성 | 변환 지점 | 변환 규칙 |
|-----------|-------------|-------------|-----------|-----------|
| targetCount·deletedCount·elapsedMs | ENT-006 | detail | 도메인→ENT(감사) | 배치 종료 시 요약을 감사 로그 detail 로 기록(OPS-003-03) |

### 사용처

| SVC 코드 | 기능 | 용도 | 사용 PROC | 비고 |
|----------|------|------|----------|------|
| SVC-007 | 배치 감사 | 결과 | PROC-402 | 사용자·서비스 대면 응답 없음 |

### 구현 가이드

- 배치 결과는 별도 상태 테이블에 저장하지 않고 감사 로그(ENT-006)에 요약 기록한다. 실패 시 다음 주기 재시도로 흡수한다(OPS-003-03).
